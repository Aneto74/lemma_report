CSV Lemma Report — Инструкция по запуску
========================================

1. Что это?
------------
CSV Lemma Report — это программа для лемматизации CSV-файлов с помощью mystem и создания отчёта в удобном формате. Не требует установки Python.

2. Что нужно для работы?
------------------------
Вам понадобятся два файла:
  - gui_lemma_report.exe
  - mystem.exe

Файл mystem можно скачать с https://tech.yandex.ru/mystem/

3. Как подготовить папку?
-------------------------
Скопируйте оба файла в одну папку. Пример:

  Папка/
    gui_lemma_report.exe
    mystem.exe

4. Как запустить программу?
---------------------------
1) Дважды кликните по файлу gui_lemma_report.exe.
2) Откроется окно программы.
3) Нажмите "Выбрать файл" и укажите исходный CSV-файл.
4) Нажмите "Запустить обработку".
5) Дождитесь завершения — итоговый файл появится рядом с исходным.

5. Важно!
----------
- Не удаляйте mystem.exe из папки с программой.
- Программа не работает без mystem.exe.
- Если путь к файлам содержит пробелы — это не проблема, программа поддерживает такие пути.

6. Возможные ошибки
-------------------
- Если появляется сообщение "mystem не создал файл лемм" — проверьте, что mystem.exe лежит рядом с программой и не занят другими процессами.

7. Вопросы и поддержка
----------------------
Если возникли вопросы или проблемы — обратитесь к разработчику или в техподдержку.

8. Для пользователей Mac и Linux
-------------------------------
Если вы используете macOS или Linux:
- Вместо exe-файла используйте скрипт lemma_report.py (требуется установленный Python 3).
- Скачайте подходящую версию mystem для вашей ОС с официального сайта: https://tech.yandex.ru/mystem/
- Поместите файл mystem (или mystem.exe для Windows) в ту же папку, что и lemma_report.py.
- Запустите обработку командой:

  python3 lemma_report.py

- Далее следуйте инструкциям в терминале или в самом скрипте.
